Key Insights


Orders peak between 12–1 PM and again from 4–7 PM, useful for shift planning.
Classic is the top-performing category by both revenue and volume.
Large pizzas make up almost half of all sales (45.9%), the clear customer favorite.
Weekly order volume varies quite a bit (from 171 to 491 orders), pointing to seasonal demand swings.
Revenue is spread fairly evenly across categories, so no single category is carrying the business.
